## ğŸš€ Getting Started

### Prerequisites
- Modern web browser
- Local development server (optional)

### Installation
1. Clone the repository:
```bash
git clone https://github.com/maheshinho10/IamMahesh.github.io.git
```

2. Navigate to the project directory:
```bash
cd IamMahesh.github.io
```

3. Open `index.html` in your browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .

# Using PHP
php -S localhost:8000
```

```css

  --secondary-color: #8b5cf6;
```

- **Personal Info**: Update content in `index.html`
- **Skills**: Modify `skillsData` object in `script.js`
- **Projects**: Update `projectsData` array in `script.js`

### Styling

- **Animations**: Adjust timing and effects in CSS
- **Responsive**: Update media queries for different breakpoints
